Condicional If

Con directivas

Status es {{statusDirectiva}}
No hay status

Nueva sintaxis

@if (statusNueva) {
Status es {{statusNueva}}
} @else {
No hay status
}

Ejemplo con else if

@if (edad) { @if (edad < 0) {
Edad no válida
} @else if (edad < 18) {
Eres menor de edad
} @else {
Eres mayor de edad
} }

Condicional Switch

Con directivas

Opción 1 seleccionada
Opción 2 seleccionada
Opción 3 seleccionada
Ninguna opción seleccionada

Nueva sintaxis

@switch (selectedOption) { @case ('option1') {
Opción 1 seleccionada
} @case ('option2') {
Opción 2 seleccionada
} @case ('option3') {
Opción 3 seleccionada
} @default {
Ninguna opción seleccionada
} }
